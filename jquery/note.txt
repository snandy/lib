jQuery.fx 模块概述
jQuery提供了一些方法可以在页面上添加一些动画，包括一些经常使用的简单的，基本的动画。当然你也可以自定义一些精巧的，复杂的动画。
基本的动画方法是animate，其它如toggle、show、hide、slideDown、slideUp、slideToggle、fadeIn、fadeOut、fadeToggle都是调用animate。

jQuery.speed

1 第一个参数speed两种形式的处理 对象和字符串，如果是对象则直接扩展它
2 对opt.duration的处理，
	$.fx.off 为true则关闭动画效果
	opt.duration为数字则直接使用
	opt.duration为字符串，判断是否在jQuery.fx.speeds中定义过，定义过则取
	最后才使用$.fx.speeds._default

genFx
1	生成动画需要的标准参数对象， 两个参数type, includeWidth。
2	仅传type时 生成如下结构对象
		{
			height: true,
			marginTop: true,
			paddingTop: true,
			marginBottom: true,
			paddingBottom: true
		}
3	includeWidth为true时，水平方向也包括。包含marginLeft, marginRight, paddingLeft, paddingRight
		{
			height: true,
			marginTop: true,
			paddingTop: true,
			marginBottom: true,
			paddingBottom: true,
			marginLeft: true,
			marginRight: true,
			paddingLeft: true,
			paddingRight: true
		}
